## 🆕 今日所学 / What I learned

- OpenAI 把 computer-use 正式装进 ChatGPT 桌面版：in-app browser、Chrome 扩展接管标签页、macOS 上有独立光标可后台操作——验证了「代理操作真实界面」已是产品级方向
- Cua (Computer-Use 2.0)：accessibility-tree 优先、像素兜底、把操作范围限定在单个窗口而不是整个桌面 → 通过率 62%→80%，token -34%。对做屏幕代理的团队，这验证了「窗口级捕获」是对的方向；常缺的一环是 accessibility 层
- Zara Zhang 的策展方法论：follow builders not influencers；把 YouTube 当人名搜索引擎（爱用的产品 → 找到 PM/创始人 → 看完他所有访谈）
- Claude Code 团队（1小时访谈）：给前沿模型的提示词要删例子、删 always/never 硬约束——每条指令必须 100% 为真；他们靠这个把系统提示词砍掉 80%。已采纳为写 skill 的新纪律
